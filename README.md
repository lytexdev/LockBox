# Lock Box

## Overview
Lock Box is a School project which is a secure password manager for storing and organizing your credentials.

## Installation

### Prerequisites
- Java 17 or higher

**Clone the repository**
```bash
git clone https://github.com/lytexdev/lockbox.git
cd lockbox
```

**Build the application**
```bash
javac *.java
```

**Run the application**
```bash
java LockBox
```
